API Access

All requests to https://api.casualchallenge.gg/v1/... require an API token. You can get a token by reaching out to Raoul, Merchant of the Void via Discord.

Motivation

This API provides machine-readable data about cards in our custom Magic: the Gathering format, Casual Challenge.

The goal is to empower tech-savvy players to build their own tools, explore new ideas, and contribute to developing the format further.
